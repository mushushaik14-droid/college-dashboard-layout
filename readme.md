# ChatGPT Pricing Card Webpage

## 1. Description

The **ChatGPT Pricing Card Webpage** is a simple HTML5 and CSS3 project that displays different ChatGPT subscription plans in a clean pricing table. The project demonstrates how to organize pricing information using tables and apply CSS styling for a professional appearance.

---

## 2. Vision / Purpose

The purpose of this project is to practice designing a professional pricing card webpage using HTML5 and CSS3. It helps in understanding tables, layouts, colors, typography, spacing, and external CSS styling.

---

## 3. Technologies Used

- HTML5
- CSS3

---

## 4. Features

- Professional header section
- Pricing comparison table
- Free, Plus, Pro, Business, and Enterprise plans
- Plan features comparison
- Upgrade and contact options
- "Learn More" links
- Footer section
