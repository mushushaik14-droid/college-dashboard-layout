# College Dashboard Layout

## 1. Description

The **College Dashboard Layout** is a simple HTML5 and CSS3 project that demonstrates a professional college dashboard interface. It includes a header, sidebar navigation, dashboard statistics, announcements, recent activities, upcoming events, and a footer section.

---

## 2. Vision / Purpose

The purpose of this project is to practice building a dashboard layout using HTML5 and CSS3. It helps in understanding webpage layouts, sidebars, sections, forms, colors, spacing, and external CSS styling.

---

## 3. Technologies Used

- HTML5
- CSS3

---

## 4. Features

- College header section
- Search bar
- Sidebar navigation menu
- Statistics cards
- Announcements section
- Recent activities section
- Upcoming events section
- Professional footer
